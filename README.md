# BAZA Trainer

Мобильное приложение для фитнеса и учёта тренировок на **Flutter**. Проект выполнен **в составе команды** (командная работа).

## Описание

BAZA Trainer помогает вести профиль, планировать и проходить тренировки, вести дневник и раздел питания. Данные синхронизируются с бэкендом **Supabase** (аутентификация, таблицы тренировок и упражнений и др.).

### Основные возможности

- **Профиль** — экран пользователя и навигация по приложению  
- **Тренировки** — программы, создание и прохождение тренировок, таймеры и прогресс  
- **Дневник** — записи и календарь  
- **Питание** — раздел учёта/планирования питания  
- **Система** — системные настройки приложения  
- **Регистрация и вход** — работа с сессией через `SessionManager` и безопасное хранение (`flutter_secure_storage`)

Интерфейс ориентирован на русскоязычных пользователей.

## Технологии

| Категория | Стек |
|-----------|------|
| Фреймворк | Flutter (Dart SDK ≥ 3.1.4) |
| Состояние / DI | `provider` |
| Бэкенд | Supabase (`supabase_flutter`) |
| Локальное хранение | `shared_preferences`, `flutter_secure_storage` |
| Медиа | `image_picker`, `permission_handler` |
| Прочее | `intl`, таймеры (`circular_countdown_timer`), `http` |

## Структура проекта

Исходный код приложения лежит в каталоге **`bazatrainer/`**.

```
bazatrainer/
├── lib/
│   ├── Presentation/     # UI: экраны входа, регистрации, тренировок, дневника и т.д.
│   ├── Domain/           # Сессия, Supabase-клиент, валидация пароля, трекер тренировок
│   ├── Data/             # Сервисы: пользователь, логин, регистрация, тренировки, хранилище
│   └── main.dart         # Точка входа BazaTrainerApp
├── assets/pictures/      # Изображения
└── test/                 # Тесты (в т.ч. виджет-тесты экрана входа)
```

## Требования

- Установленный [Flutter](https://docs.flutter.dev/get-started/install) (стабильный канал, совместимый с SDK из `pubspec.yaml`)
- Для iOS: Xcode (только на macOS)
- Для Android: Android SDK / эмулятор или устройство

## Запуск

```bash
cd bazatrainer
flutter pub get
flutter run
```

Укажите целевую платформу при необходимости, например:

```bash
flutter run -d chrome    # веб
flutter run -d windows   # Windows
```

## Тесты

```bash
cd bazatrainer
flutter test
```

## Подключение к Supabase

URL проекта и anon key задаются в `bazatrainer/lib/Domain/supabaseCli.dart`. Для продакшена и публичных репозиториев **рекомендуется** не хранить ключи в коде, а подставлять их через `--dart-define`, файлы окружения или CI-секреты и при утечке **ротировать** ключи в консоли Supabase.
